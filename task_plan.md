# Scout 增强计划

## 总览

7 个阶段，按依赖关系和难度排序。每个阶段独立可交付，完成后可立即使用。

## 阶段 1：Docker 部署 ⭐ — 暂缓
- 推出公开版本时再做

- [x] Startpage 引擎（HTML 解析，curl_cffi）
- [x] Yandex 引擎（HTML 解析）
- [x] DuckDuckGo News 引擎（HTML 解析，curl_cffi）

## 阶段 3：MCP Resources ⭐⭐
- [ ] cache://page/{url} — Agent 可直接读取已缓存页面
- [ ] cache://stats — 缓存统计信息
- [ ] engines://list — 引擎列表资源

## 阶段 4：结果质量优化 ⭐⭐
- [ ] 域名信誉评分（白名单/黑名单）
- [ ] 新鲜度加权（越新越高分）
- [ ] 多引擎命中加权（被更多引擎命中的结果更高分）
- [ ] 页面质量信号（标题长度、摘要质量、URL 结构）
- [ ] RRF 融合参数可配置

## 阶段 5：搜索建议 ⭐⭐⭐

- [ ] 查询改写（自动优化查询词）

## 阶段 6：结果摘要 ⭐⭐⭐
- [ ] 多源内容对比摘要
- [ ] 关键信息提取
- [ ] 可信度评估

## 阶段 7：分布式缓存 ⭐⭐
